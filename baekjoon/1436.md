# 1436: 영화감독 숌

### 문제 정보
- **문제 링크**: [https://www.acmicpc.net/problem/1436](https://www.acmicpc.net/problem/1436)

---

### 나의 풀이

#### 💡 핵심 아이디어
- 가장 작은 수 666부터 시작하여 숫자를 1씩 증가시키면서, 각 숫자에 `666`이 포함되어 있는지 검색하는 `완전 탐색`으로 해결
  1. 현재 카운트를 0으로 초기화
  2. 탐색을 시작할 숫자를 666으로 초기화
  3. 카운트가 입력 N과 같아질 때까지 반복
  4. 반복문 내에서 현재 숫자를 문자열로 변환
  5. 문자열 안에 `666`이 포함되어 있는지 확인
  6. 포함되어 있다면 카운트 1 증가
  7. 카운트 = N, 현재 카운트롤 출력 및 반복문 종료
  8. num++, 다음 숫자를 검색

#### 🤔 어려웠던 점 / 배운 점
- N이 커질수록 666 수 규칙이 복잡해지기 때문에, 단순히 N - 1 에 666을 붙이는 방식으로는 문제를 풀 수 없었다

####  실행 결과

### 소스 코드
```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());
        int count = 0;
        int num = 666;
        
        while (true) {
            if (String.valueOf(num).contains("666")) {
                count++;
            }
            if (count == N) {
                System.out.println(num);
                break;
            }
            num++;
        }
    }
}
```
