# 1193: 분수찾기

### 문제 정보
- **문제 링크**: [https://www.acmicpc.net/problem/1193](https://www.acmicpc.net/problem/1193)

---

### 나의 풀이

#### 💡 핵심 아이디어
- 분수들이 나열되는 순서의 규칙성
  - 순서 `X`에 해당하는 분수 찾기

#### 🤔 어려웠던 점 / 배운 점
- `X`가 정확히 대각선 라인의 끝 번호에 걸칠 때와, 다음 라인으로 넘어가는 경계 판단

####  실행 결과
- **메모리**: 18428 KB
- **시간**: 188 ms

---

### 소스 코드
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int X = sc.nextInt();
        sc.close();

        int line = 1; 
        int count = 0;

        while (count + line < X) {
            count += line;
            line++;
        }

        int line_num = line;
        
        int position = X - count;

        int numerator;
        int denominator;

        if (line_num % 2 == 1) {
            numerator = line_num - position + 1;
            denominator = position;

        } else {
            numerator = position;
            denominator = line_num - position + 1;
        }

        System.out.println(numerator + "/" + denominator);
    }
}
```
