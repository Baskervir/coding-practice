# 2566: 최댓값

### 문제 정보
- **문제 링크**: [https://www.acmicpc.net/problem/2566](https://www.acmicpc.net/problem/2566)

---

### 나의 풀이

#### 💡 핵심 아이디어
- 2차원 배열 완전 탐색
  - 변수 선언
    - 최댓값 `max`, 최댓값의 행 `maxRow`, 열 `maxColum` 변수 선언
    - `max`의 초기값은 입력으로 들어올 수 있는 가장 작은 값보다 더 작은 값 `-1`로 하면 더 안전
  - 2중 반복문을 이용한 완전탐색
    - 모든 원소를 순회하며 탐색
  - 비교 및 업데이트
    - 현재 확인하는 원소가 지금까지 기록된 `max`보다 크면
      - `max`값을 현재 원소로 교체
      - 현재 `max`값에 해당하는 `maxRow`, `maxColum`을 각각 교체한다 (업데이트)
  - 출력

#### 🤔 어려웠던 점 / 배운 점
- 크기에 대한 사전 정보 없이 고정된 9X9 격자 데이터가 주어진다
  - `StringTokenizer st = new StringTokenizer(br.readLine());` 로 한 줄을 먼저 읽으려 한게 문제
    - `NullPointerException`
    - `NoSuchElementException`

####  실행 결과
- **메모리**: 15940 KB
- **시간**: 112 ms

---

### 소스 코드
```java
import java.util.*;
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st;
        
        int[][] matrix = new int[9][9];
        int max = 0;
        int maxRow = 0;
        int maxColum = 0;
        
        for (int i = 0; i < 9; i++) {
            st = new StringTokenizer(br.readLine());
            for (int j = 0; j < 9; j++) {
                matrix[i][j] = Integer.parseInt(st.nextToken());
            }
        }
        
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                if (matrix[i][j] > max) {
                    max = matrix[i][j];
                    maxRow = i;
                    maxColum = j;
                }
            }
        }
        
        System.out.println(max);
        System.out.println((maxRow + 1) + " " + (maxColum + 1));
    }
}
```
